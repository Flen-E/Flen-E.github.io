### 64번

**IP 프로토콜**  
+ Header Length : IP 프로토콜의 헤더 길이를 32비트 워드 단위로 함
+ Packet Length(16 bits) : IP 헤더 및 데이터를 포함한 IP 패킷 전체길이를 바이트 단위로 길이를 표시(2^16 -1)
+ Time to Live: 송신 호스트가 패킷 전송전 네트워크에서 생존할수 있는 시간 지정한것
+ Version Number : IP 프로토콜의 버전번호
